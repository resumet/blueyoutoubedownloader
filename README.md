# Blue YouTube Downloader Website

GitHub Pages source for the Blue YouTube Downloader distribution page.

## SEO focus keyword
- 유튜브 MP3 다운로드

## Deployment
1. Push this repository to GitHub.
2. In GitHub repository settings, enable GitHub Pages from the `main` branch root.
3. Replace the Google Drive button URL in `index.html`.
